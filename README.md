# Laundry Service Website
This is a laundry service website made using HTML, CSS and JavaScript.

In this website users can:
* View available laundry services
* Add services to the cart
* Remove services from the cart
* See the total amount
* Book laundry services
* Receive booking confirmation through EmailJS
* Subscribe to the newsletter

## Technologies Used
* HTML
* CSS
* JavaScript
* EmailJS

## How to Run the Project
1. Download the project files.
2. Open the project folder in VS Code.
3. Install the Live Server extension if not already installed.
4. Open the `index.html` file.
5. Right click and select **Open with Live Server**.
6. The website will open in your browser.

## Project Structure
* `index.html` - Structure of the website
* `style.css` - Styling of the website
* `script.js` - Functionality and validations

## Features Implemented
* Add and remove items from cart
* Dynamic total amount calculation
* Form validation
* Newsletter subscription validation
* Responsive design for mobile and tablet screens
* EmailJS integration for booking confirmation
